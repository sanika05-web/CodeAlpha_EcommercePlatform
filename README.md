# CodeAlpha E-commerce Website

A simple e-commerce web application built using Django.

# Features
- User Signup and Login
- Product Listing
- Buy Product (Login Required)
- My Orders Page
- Modern Dark UI

# Technologies Used
- Python
- Django
- HTML
- CSS

# How to Run the Project
1. Activate virtual environment
2. Run migrations using `python manage.py migrate`
3. Start server using `python manage.py runserver`
4. Open browser and go to `http://127.0.0.1:8000/`
